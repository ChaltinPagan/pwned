# Pwned

Find out if your password has been compromised in a data breach.

## Features

* [Have I Been Pwned API](https://haveibeenpwned.com/API/v2)
* Responsive Web Design
* Create an account

## Tech

Client Side
* React JS
* Font Awesome Icons

Server
* Node
* Express
* PostgreSQL database

## Installation

1. Clone the master branch of this repository.
2. Run the client side.
    * Open a terminal
    * `cd client`
    * `npm start` or `yarn start`
3. Seed the database using PostgreSQL.
    * Open a new terminal.
    * `cd db`
    * Use the SQL file in this folder.
4. Run the server.
    * Open a new terminal.
    * `npm start`
